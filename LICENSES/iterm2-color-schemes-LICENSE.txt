iTerm2-Color-Schemes — https://github.com/mbadolato/iTerm2-Color-Schemes

Forge bundles a curated subset of terminal color themes (in Resources/themes/)
imported from this repository. Each theme file retains a `# Source:` header
pointing at the upstream commit. The original collection license follows
verbatim. Note the final paragraph: the license for each individual theme
belongs to the original theme author — see the upstream CREDITS.md for
per-theme attribution.
