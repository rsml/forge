# Theme Acknowledgments

Forge bundles a curated subset of themes from [mbadolato/iTerm2-Color-Schemes](https://github.com/mbadolato/iTerm2-Color-Schemes). Per that repository's license stance: *"Schemes are not subject to copyright unless otherwise stated."* The bundled `.conf` files include a pinned commit SHA in their attribution header so the upstream state is auditable.

## Notes on trademarks and brand marks

Some bundled themes (Dracula in particular, also Nord and Solarized) carry brand identity and/or trademarks held by their authors. Forge ships them unmodified and credits the original authors. Forge does not claim ownership of, or extend, any third-party theme.

## Reporting attribution issues

If you author one of these themes and want a change to this attribution, open an issue at the [Forge GitHub repo](https://github.com/rsml/forge).
